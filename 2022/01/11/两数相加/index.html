两数相加

题目描述见链接：(https://leetcode-cn.com/problems/add-two-numbers)

解题思路：两个非负整数相加，相加的和除于10，余数作为新的链表的值，产生的进位加到下一次的运算中。这里两个的链表不一定一样长，对于不一般长的链表，短的链表用0补齐。如果链表最后产生了进位，则将进位补在新链表的末尾。

 public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode pre=new ListNode(0);
        ListNode cur=pre;
        int carry=0;
        while(l1!=null||l2!=null){
            int temp1=l1==null? 0:l1.val;
            int temp2=l2==null? 0:l2.val;
            int sum=temp1+temp2+carry;

            carry=sum/10;
            int temp=sum%10;
            cur.next=new ListNode(temp);
            cur=cur.next;//这里需要将指针后移，不然你得到的就是一个值，而不是链表。因为每次修改的就是这个结点的值
            if(l1!=null){
                l1=l1.next;
            } 
            if(l2!=null){
                l2=l2.next;
            }
        }
        if(carry==1){
            cur.next=new ListNode(1);
        }
        return pre.next;
    }
